# jlu-webvpn-assistant

吉林大学 WebVPN 助手。更方便的使用 WebVPN，允许访问其他校内地址。

## 通过 Greasy Fork 安装

https://greasyfork.org/zh-CN/scripts/395271

## 构建

```shell
npm run build
```

安装 `dist/jlu-webvpn-assistant.user.js` 文件。
